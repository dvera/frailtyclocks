This webapp deploys machine learning-derived models to predict age and lifespan from the frailty assessment of mice aged 22 months or older. These clocks are based on random forest regressions, trained on a naturally aging population of male C57Bl6/NIA mice.

FRIGHT age (Frailty Inferred Geriatric Health Timeline) is a measure of a mouse's apparent chronological age.
The AFRAID clock (Analysis of Frailty and Death) is a prediction of a mouseâ€™s life expectancy.

The clocks can be run on this webapp by uploading your frailty data in the box on the left, or on users' computers in Python with the models available on our repository.


Development of frailty-based clocks: (a) Box and whisker plots for frailty index (FI) score for mice from 21 to 36 months of age. (b) Random forest regression of the individual frailty index items for chronological age on testing dataset (FRIGHT age). (c) Random forest regression of the individual frailty index items for life expectancy on testing datasets, plotted against actual survival by age group (AFRAID clock).

Using the clocks:

  1. Measure frailty data (31 items) in mice as first described by Whitehead et al. 2014 and further described by Schultz et al., 2020. We recommend recording data using our scantron score sheet.
  2. Upload score sheets to formread.org or similar website to produce data in Excel format. Contact alice_kane@hms.harvard.edu for a formread.org template.
  3. Calculate weight variables (recent and total weight change, threshold weight change, and body weight standard deviation) as described in Schultz et al., 2020.
  4. Put your data in the same format as the example datasheet with each row representing an individual mouse's frailty variables. The variable names must exactly match the names in the example datasheet. If group names are included, the sheet must be sorted by group.
  5. Upload your datasheet on left to calculate FRIGHT age and AFRAID scores for each mouse using the "Choose File" and "Analyze" buttons. (If an error is generated, most likely the .csv file is formatted incorrectly.) The resulting values will be added to your datasheet and can be downloaded using the link on the left. A preview of your data can be viewed under the Analyzed Data tab.
